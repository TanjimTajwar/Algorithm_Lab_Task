Euler's Method Problems in Numerical Analysis

Problem 1:
Solve the initial value problem y' = x + y, y(0) = 1 using Euler's method with h = 0.1 to find y(0.5).

Problem 2:
Given the differential equation y' = 2x - y, y(0) = 1, use Euler's method with step size h = 0.2 to approximate y(1).

Problem 3:
For the differential equation y' = y^2 - x, y(0) = 0.5, apply Euler's method with h = 0.1 to find y(0.3).

Problem 4:
Using Euler's method with h = 0.05, solve y' = sin(x) + cos(y), y(0) = 0 to approximate y(0.2).

Problem 5:
For the initial value problem y' = e^x - y, y(0) = 1, use Euler's method with h = 0.1 to find y(0.4).

Problem 6:
Solve y' = x^2 + y^2, y(0) = 0 using Euler's method with h = 0.1 to approximate y(0.3).

Problem 7:
Given y' = xy + 1, y(0) = 1, apply Euler's method with h = 0.2 to find y(0.6).

Problem 8:
For the differential equation y' = y - x^2, y(0) = 1, use Euler's method with h = 0.1 to approximate y(0.5).

Problem 9:
Using Euler's method with h = 0.05, solve y' = x + y^2, y(0) = 0 to find y(0.2).

Problem 10:
For the initial value problem y' = 2y - x, y(0) = 1, use Euler's method with h = 0.1 to approximate y(0.4).

Note: For each problem:
1. Show all steps of calculation
2. Compare your numerical solution with the exact solution (if possible)
3. Calculate the error at each step
4. Comment on the accuracy of the method
